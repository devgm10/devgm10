Construyo software de principio a fin: del componente al contenedor, y del contenedor a la nube.

Más de **5 años** entregando soluciones integrales. ### Proyectos

**[landing-page](https://github.com/devgm10/landing-page)** — Mi portfolio, construido como un producto en producción.

Vue 3 + Docker + Terraform + GitHub Actions. Despliegue en EC2 **sin acceso SSH** (vía AWS SSM), autenticación en CI por **OIDC** (sin credenciales estáticas), análisis de calidad con SonarCloud, escaneo de seguridad con Snyk y observabilidad con Prometheus + Grafana. El microservicio de contacto corre en Lambda, con rate limiting y verificación anti-bot server-side.

**PLANIFLY AI** — Plataforma SaaS financiera con IA.

Automatiza el registro y análisis de gastos personales. Django + DRF + Channels + Redis, con motor de IA e integración por WhatsApp. Proyecto propio.

**PCG AGRO** · **HARVEST** · **SCRAPING AGRO** — Plataformas corporativas en Agrovision Corp.

ERP agrícola a escala global · plataforma geoespacial de monitoreo de cosecha en tiempo real · plataforma ETL/ELT para centralizar grandes volúmenes de datos agrícolas.
